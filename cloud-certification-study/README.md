## 📏 Study Rules (read these once, follow them always)

1. **Every topic must have 5 things:** a concept note, a service mapping (AWS↔Azure), a hands-on lab, common exam traps, and flashcards.
2. **Every lab ends with a CLEANUP step.** Cloud costs money. Delete what you create. Log it in the lab tracker.
3. **Every cert study session includes:** a beginner explanation + a real-world example + a few exam-style questions.
4. **Every exam gets a "Last 3 Days Revision Checklist."** No new topics in the last 3 days — only review.
5. **For every cloud service, learn the AWS vs Azure equivalent.** Knowing both makes you twice as employable.
