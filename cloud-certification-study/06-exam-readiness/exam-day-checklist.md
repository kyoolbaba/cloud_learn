# 🗓️ Exam Day Checklist

## The night before
- [ ] **No new topics.** Light review of `weak-topic-log.md` + flashcards only.
- [ ] Confirm exam time + time zone
- [ ] (Online) Re-run the proctoring system check
- [ ] Prepare ID(s) required
- [ ] Clear your desk/room (online proctoring rules: nothing on walls/desk)
- [ ] Sleep early 😴

## 1–2 hours before
- [ ] Eat + hydrate (but not too much liquid)
- [ ] Quick flashcard skim — confidence, not cramming
- [ ] Close all apps; restart computer (online)
- [ ] Quiet, well-lit room; phone away
- [ ] Bathroom break

## During the exam
- [ ] Read each question **twice**; underline keywords mentally (*most cost-effective, HA, least overhead, real-time*)
- [ ] Eliminate clearly-wrong options first
- [ ] For "choose 2/3," judge each option independently
- [ ] **Flag and skip** hard ones; don't burn time
- [ ] Watch the clock (~budget time/question)
- [ ] Don't change answers without a clear reason
- [ ] Review all flagged questions before submitting

## Common keyword → answer instincts
| Keyword | Lean toward |
|---|---|
| "most cost-effective" + interruptible | Spot |
| "highly available" | Multi-AZ / zone-redundant |
| "least operational overhead" | Managed / serverless |
| "decouple" | Queue (SQS / Service Bus) |
| "real-time / low latency" | Streaming / cache / endpoint |
| "global low latency" | CDN (CloudFront / Front Door) |
| "audit who did what" | CloudTrail / Activity Log |

## After the exam
- [ ] Note the result + how it felt
- [ ] Update `master-certification-tracker.md`
- [ ] (If passed) share/celebrate; plan next cert
- [ ] (If not) log weak areas, schedule retake, no shame — it happens
