# 🧭 Recommended Certification Path (Data Scientist / Analytics)

Three paths below. **If unsure, choose Path C** — it's built specifically for a data-science career and balances both clouds.

---

## Path A — AWS-first
For: you want depth in one cloud, AWS is common at your target jobs.

| # | Certification | Code | Why |
|---|---|---|---|
| 1 | AWS Cloud Practitioner *(skip if already comfortable)* | CLF-C02 | Vocabulary + safety |
| 2 | AWS AI Practitioner | AIF-C01 | AI/ML concepts on AWS |
| 3 | AWS Solutions Architect – Associate | SAA-C03 | The "how cloud fits together" cert |
| 4 | AWS Data Engineer – Associate | DEA-C01 | Pipelines, lakes, warehouses |
| 5 | AWS Machine Learning Engineer – Associate | MLA-C01 | Train + deploy + monitor ML |
| 6 | AWS Generative AI Developer – Professional | AIP-C01 | Production GenAI / RAG / agents |
| 7 | *(Optional)* AWS Security – Specialty | SCS-C03 | Hardening + compliance |

---

## Path B — Azure-first
For: your company/target jobs use Microsoft stack.

| # | Certification | Code | Why |
|---|---|---|---|
| 1 | Azure Fundamentals | AZ-900 | Vocabulary + safety |
| 2 | Azure Data Fundamentals | DP-900 | Relational/non-relational/analytics |
| 3 | Azure AI Fundamentals | AI-900 | AI workloads + GenAI basics |
| 4 | Azure Administrator | AZ-104 | Core hands-on cloud ops |
| 5 | Azure Developer **or** Azure Security | AZ-204 / AZ-500 | Pick by job role |
| 6 | Azure Solutions Architect Expert | AZ-305 | Enterprise design |
| 7 | *(Optional)* Cosmos DB Specialty **or** DevOps Expert | DP-420 / AZ-400 | Specialize |

---

## ⭐ Path C — Best Practical Path for a Data Scientist (RECOMMENDED)
Learn each layer in **both** clouds before moving up. This is the most employable sequence for analytics/DS.

| Stage | Goal | AWS | Azure |
|---|---|---|---|
| 1. Cloud basics | Speak cloud, stay safe | Cloud Practitioner (CLF-C02) | AZ-900 |
| 2. Data basics | Understand data services | Data Engineer Associate (DEA-C01) | DP-900 |
| 3. AI basics | Understand AI workloads | AI Practitioner (AIF-C01) | AI-900 |
| 4. Architecture | Design systems | Solutions Architect Assoc (SAA-C03) | AZ-305 (basics) |
| 5. ML deployment | Ship models | ML Engineer Associate (MLA-C01) | — (use Azure ML labs) |
| 6. GenAI production | Build real GenAI | Generative AI Dev Pro (AIP-C01) | — (use Azure OpenAI labs) |
| 7. Security basics | Protect it all | Security Specialty (SCS-C03) | **or** AZ-500 |

### Suggested calendar for Path C (realistic, part-time ~8–10 hrs/week)

| Months | Focus | Certs targeted |
|---|---|---|
| 1 | Cloud basics | CLF-C02, AZ-900 |
| 2–3 | Data basics | DP-900, DEA-C01 |
| 4 | AI basics | AIF-C01, AI-900 |
| 5–6 | Architecture | SAA-C03 |
| 7–8 | ML deployment | MLA-C01 |
| 9–10 | GenAI | AIP-C01 |
| 11+ | Security (optional) | SCS-C03 or AZ-500 |

> Adjust speed to your time. The **order** matters more than the dates.
