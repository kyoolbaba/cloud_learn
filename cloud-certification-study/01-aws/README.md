# 🟧 AWS Certifications

One Markdown file per certification, grouped by level. Each file follows the same 17-section template so you always know where to look.

## Folders
| Folder | Certs |
|---|---|
| `foundational/` | Cloud Practitioner (CLF-C02), AI Practitioner (AIF-C01) |
| `associate/` | Solutions Architect (SAA-C03), Developer (DVA-C02), Data Engineer (DEA-C01), ML Engineer (MLA-C01), CloudOps (SOA-C03) |
| `professional/` | Solutions Architect Pro (SAP-C02), DevOps Pro (DOP-C02), Generative AI Dev Pro (AIP-C01) |
| `specialty/` | Advanced Networking (ANS-C01), Security (SCS-C03) |
| `labs/` | Reusable lab guides + cleanup checklists |

## Recommended AWS order for a Data Scientist
CLF-C02 → AIF-C01 → SAA-C03 → DEA-C01 → MLA-C01 → AIP-C01 → (optional) SCS-C03

## Every cert file contains
1. Name · 2. Exam code · 3. Level · 4. Best for role · 5. Prerequisites · 6. Core topics · 7. Services to learn · 8. Hands-on labs · 9. 2-week plan · 10. 4-week plan · 11. 8-week plan · 12. Mock exam strategy · 13. Final revision checklist · 14. Notes template · 15. Mistake log · 16. Flashcards · 17. Official docs/resources

> 💸 Reminder: every lab ends with a **Cleanup** step. Track it in `04-trackers/lab-completion-tracker.md`.
