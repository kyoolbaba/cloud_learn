# Microsoft Certified: Azure AI Fundamentals

> 🍼 **Beginner explanation:** Introduces AI concepts and Azure's AI services — machine learning, computer vision, natural language, and generative AI — at a beginner level. Great for a data scientist starting on Azure.
>
> 🌍 **Real-world example:** Build (conceptually) a system that reads invoices (vision), understands support emails (NLP), and answers questions with generative AI — using Azure AI services.
